# 🖼️ Image Gallery

A responsive and modern Image Gallery built using HTML, CSS, and JavaScript. It allows users to browse images by category with a clean, user-friendly interface.

## ✨ Features

- Responsive design for desktop and mobile
- Category filter (Nature, Cars, Animals)
- Smooth hover effects
- Modern UI
- Clean and organized code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Image-Gallery/
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
